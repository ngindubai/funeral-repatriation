# route_data/

Per-origin JSON files for the route generator (Engine 1).

Each file covers one origin country and all its destination corridors.
The generator reads these files to produce full-content route pages.

See site/data/route_data/*.json for the actual data files.

Note: This README lives at repo root, NOT inside site/data/route_data/,
because Hugo parses everything in site/data/ as data and cannot handle .md files.
